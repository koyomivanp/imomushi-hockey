這った跡が、カベになる。

芋虫を動かして葉っぱ（パック）を相手ゴールへ。通常移動で這った跡が体節（壁）になり、コートを自分色に染めていく 2D エアホッケーです。

Shift ダッシュなら体節は出ず、勢いを保って壁をすり抜けられます。先取 3 点で勝利。CPU 対戦（3 段階）とローカル 2 人対戦に対応。

特徴

  • 這う = 壁を作る、独自の対戦ルール
  • ダッシュ貫通で攻防が一気に入れ替わる
  • 森テーマ UI・シネマティックな画面遷移
  • CPU 難易度 3 段階 / 2P 同画面

操作

場面キー
タイトル↑↓ 選択 / Space 決定 / M BGM / F 全画面
CPU 難易度A D 選択 / 3・4・5 で直接選択 / Space 決定
P1WASD 移動 / Shift ダッシュ
P2矢印 移動 / Shift ダッシュ
