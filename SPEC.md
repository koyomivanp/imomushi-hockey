# エアホッケーが終わった — 仕様 v0.6

> **注意:** この文書は初期設計メモです。現在の実装（v0.1.0）は先取3点・残機なし・軌跡壁が中心です。最新の遊び方は [README.md](README.md) を参照してください。

Jam版：キーボード2P・ローカル対戦。実装は Pygame 推奨（毎フレーム物理向き）。

---

## 一言ピッチ

普通のエアホッケーが、アイテムとパック増殖でだんだんおかしくなるバカ対戦ゲー。

**Art direction:** テーブル全面・背景なし。筐体（エアホッケーテーブル）だけが画面いっぱい。

**これは NOT:** 本気の e スポーツ、リアル氷物理、オンライン対戦、時間制限勝敗。

---

## 勝敗

| 条件 | 結果 |
|------|------|
| スコアが先に **10点** に到達 | 即勝ち |
| 相手の残機が **0** | 即勝ち |

- **時間制限なし**（タイマーは任意表示のみ）
- 同時成立時は **スコア到達を優先**

### 残機

- 各プレイヤー **3機** で開始
- 減る条件は **自ゴールのみ**（下記）
- スター命中では **減らない**

### スコア

| イベント | 得点 |
|----------|------|
| 通常ゴール | +1（ゴールした側） |
| デカパックが相手ゴールに入る | +3（デカパックを打ち込んだ側） |

---

## 残機が減る条件

1. **自ゴール** — パックが自分側のゴールに入った → **残機 -1**
2. **光線銃** — ノックバックされたパックが自ゴールに入った → **残機 -1**

※ 残機 0 で即試合終了。

## 画面構成

```
P1 ♥♥♥   4 - 2   ♥♥♡ P2
┌─────────────────────────────┐
│         [ テーブル ]          │  ← 画面の 80% 以上
│    ゴール上（P2）             │
│         ○  ★  ○              │
│    ゴール下（P1）             │
└─────────────────────────────┘
```

- 解像度目安: **800×600**
- 背景: 単色 or 極薄グラデ（装飾なし）
- テーブル: 横長矩形。左右に壁、上下にゴール（穴）
- ゴール幅: テーブル幅の **40%**（中央に空き）
- UI: 残機・スコア・（任意）経過時間・アイテム取得時の1行テキスト

## 操作

| P1（下側） | P2（上側） |
|------------|------------|
| `A` / `D` 左右 | `←` / `→` 左右 |
| `W` / `S` 上下 | `↑` / `↓` 上下 |
| `Space` タイトル開始 / リザルト再戦 | 同左 |
| `Esc` ポーズ / タイトルへ | 同左 |

### 移動

- **4方向**、斜め同時押し可（ベクトル正規化で速度一定）
- 移動範囲: **テーブル全域**（中央ライン越え OK、ゴール前キャンプ OK）
- ラケット同士が重なったら **押し出し**（完全重なり防止）
- 通常速度: **300 px/s**

### ラケット

- 幅: テーブル幅の **18%**（巨大化時 2倍）
- 形状: 円 or 角丸矩形（当たり判定は円推奨）

---

## パック

| 項目 | ルール |
|------|--------|
| 試合開始 | 中央に **1個** |
| ゴール後 | 全消去 → **1.5秒** 演出 → 中央 **1個** 再配置 |
| 場内上限 | **8個** |
| 増殖 | **デカパックのみ**（時間経過・通常ゴールでは増えない） |

### 物理（初期案）

| パラメータ | 値 |
|------------|-----|
| 半径 | 12px |
| 最大速度 | 500 px/s |
| 最低速度 | 80 px/s（詰まり防止） |
| 壁・ラケット反射 | 弾性係数 1.0 |
| ラケット→パック | ラケット移動方向成分を加算 |
| 摩擦 | なし |
| パック同士 | 弾性衝突（円形） |

## 試合フロー

```
タイトル → 3, 2, 1 → 本戦 → リザルト → Space で再戦
```

### タイトル

- タイトル: **エアホッケーが終わった**
- サブ: `P1: WASD  P2: 矢印  Space: 開始`

### カウントダウン中

- ラケットは動かせる
- パックは動かない

## アイテム（共通ルール）

- 初回落下: 試合開始 **12秒後**
- 以降: **10秒ごと** に 1個
- 地上最大 **2個**
- 落下位置: テーブル内ランダム（上から降りてくる演出可）
- 取得: ラケット接触 → **即発動**（ストックなし）
- 5種 **均等ランダム**

---

## アイテム（5種）

### 1. デカパック `mega_puck`

- 金色のデカパック **1個** をテーブルに追加（パック増殖）
- 当たり判定 **1.8倍**
- **相手ゴール** に入ると取得者 **+3点**
- **自ゴール** に入るとその側 **残機 -1**、得点なし
- ゴール or 消滅まで存在（単体1個のみ）

### 2. スター `star`（場崩し・残機減なし）

- 取得者のラケットが **虹色**、**5秒間**
- 移動速度 **1.2倍**（360 px/s）
- **相手ラケット** に接触:
  - **大ノックバック**（当たった方向へ強く弾く）
  - 約 **1.2秒** 操作不能
  - **残機は減らさない**
- パックに当たる → 大きく弾くだけ
- 演出: `STAR!` ＋ 虹色トレイル

### 3. お助けキャラ `assist`

- 取得者側に **コインラバー** が出現、**8秒間** 自動行動
- テーブルを走り回り、パックに触れると **相手ゴール寄り** に強く弾く
- 味方ラケットに害なし
- v0 は **1種固定**
- 演出: `HELP!`

### 4. ホッケー巨大化 `big_paddle`

- 取得者のラケット幅 **2倍**、**5秒間**
- 移動速度はそのまま
- 演出: `BIG!`

### 5. 光線銃 `ray_gun`

- **5秒間**、ラケット位置から連射（約 **0.15秒** 間隔）
- 照準: **最後に動いた方向**（停止中は相手方向）
- パック・相手ラケットに命中 → **ノックバック**
- ノックバックされたパックが **自ゴール** に入る → その側 **残機 -1**
- ラケット直撃では残機減らない
- 演出: ピュピュッ ＋ ヒットスパーク

## アイテムとパックの関係

| アイテム | パックへの影響 |
|----------|----------------|
| デカパック | +1（金色・+3点） |
| スター | パックを大ノックバック |
| お助け | パックを相手ゴール方向へ弾く |
| 巨大化 | なし |
| 光線銃 | 既存パックをノックバック |

| イベント | 表示 / SE |
|----------|-----------|
| 通常ゴール | `GOAL!` +1 |

## バランス調整の優先順位（実装後）

1. 勝利スコア（10 → 7 / 15）
2. スターのノックバック強度・硬直時間
3. 光線銃の連射間隔・ノックバック
4. デカパックの出現頻度（間接的）
5. ラケット速度・パック上限

**合格ライン:** 2人で 3 試合やって、最低 1 回は笑う。
