# スプライト

| ファイル名 | 用途 |
|-----------|------|
| `leaf_puck.png` | 葉っぱパック（透過処理あり） |

芋虫の体・顔は `caterpillar_art.py` のプロシージャル描画を使用しています。

## 葉っぱ画像の差し替え

`leaf_puck.png` を同フォルダに置くと自動で読み込まれます。黒背景は起動時にアルファ透過されます。

プロンプト例（俯瞰）:

```
top-down stylized green leaf with veins, glossy cartoon, solid black background
```
