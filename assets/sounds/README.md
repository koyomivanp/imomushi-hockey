# サウンド（任意）

このフォルダにファイルを置くと、内蔵のビープ音より優先して再生されます。

| ファイル名 | 用途 |
|------------|------|
| `bgm_title.mp3` / `.ogg` / `.wav` | タイトル BGM |
| `bgm_battle.mp3` / `.ogg` / `.wav` | 対戦 BGM |
| `wall_bounce.wav` | 壁反射 SE |
| `goal.wav` | ゴール SE |
| `countdown.wav` | カウントダウン SE |
| `start.wav` | 試合開始 SE |

ファイルが無い場合はプログラム内で生成した簡易音が使われます。
